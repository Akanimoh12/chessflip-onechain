# ChessFlip Move Contract

This directory contains the Move smart contracts for the ChessFlip application on One-Chain.

## Project Structure

- `sources/` - Move source code files
- `tests/` - Move test files
- `build/` - Compiled Move packages (generated)
- `Move.toml` - Move package manifest

## Building

```bash
move build
```

## Testing

```bash
move test
```

## Development Notes

- **Language**: Move
- **Blockchain**: One-Chain
- **Purpose**: Smart contracts for chess game logic and asset management
